# Unity-AI-Racing-cars-with-waypoints
This project is an android based car racing game made with unity

Implemented
============

* waypoints for AI cars to follow
* Track Selection
* Time laps challenge
* Racing Challenge
* Tracking time in each laps
* Two tracks and four AI cars
